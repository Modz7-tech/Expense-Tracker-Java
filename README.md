# 💰 Student Expense Tracker (Java CLI Project)

## 📌 Overview

The **Student Expense Tracker** is a command-line based Java application designed to help students manage their daily expenses efficiently. It allows users to add, view, and delete expenses, categorize them, and store data permanently using file handling.

This project was developed as part of a university requirement with a focus on **incremental development using GitHub version control**.

---

## 🎯 Problem Statement

Managing daily expenses is a common challenge for students. Without proper tracking, it becomes difficult to understand spending patterns or control unnecessary expenses.

This project aims to provide a **simple, lightweight, and effective solution** for tracking expenses using core Java concepts.

---

## 🚀 Features

* ✅ Add expenses with name, amount, and category
* ✅ View all recorded expenses
* ✅ Calculate total spending
* ✅ Delete specific expenses
* ✅ Persistent storage using file handling
* ✅ Clean and interactive CLI interface

---

## 🛠️ Technologies Used
